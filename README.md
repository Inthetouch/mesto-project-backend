# Проект Mesto (Бэкенд)

Репозиторий для бэкенд-части приложения "Место", в котором студенты делятся фотографиями интересных мест. 
Проект реализует API для управления пользователями и карточками, а также базовую логику авторизации и обработки ошибок.

## Технологии
- Node.js
- Express
- MongoDB (Mongoose)
- TypeScript
- ESLint (Airbnb Style Guide)

## Установка и запуск

1. Установка зависимостей:
   ```bash
   npm install
   ```
2. Убедитесь, что у вас запущен локальный сервер MongoDB по адресу: ```mongodb://127.0.0.1:27017/mestodb```
3. Запустите проект в режиме разработки:
   ```bash
   npm run dev
   ```

### Доступные скрипты
- ```npm run start``` — запуск сервера
- ```npm run dev``` — запуск сервера в режиме разработки
- ```npm ruin build``` — сборка проекта
- ```npm run lint``` — запуск проверки кода с помощью ESLint
